Career Connector Global Locations

Simplify Workforce has currently been implemented in over 20 countries and is being implemented in an additional 130 countries this year. Simplify Workforce being a technology company does not necessarily have local offices in each of these countries. All the implementations are cloud based using local rules (example GDPR) and support is provided from global support centers across time zones. Simplify Workforce will be able to support Microsoft in any country due to the same approach. We will hire local resources if needed to support the coordination/job seeker process.
